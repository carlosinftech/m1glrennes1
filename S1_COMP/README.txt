Ce travail a été créé pour le cours Analyse et Conception Formelles
a l'université de Rennes 1. 
Le travail a consisté en créer un prouveur qui pourrait verifier 
des proprietes sur les transactions entre un vendeur et un marchand .
Les proprietés ont été definis par les instructeurs et le programme qui 
gérait les transactions nous a été donné pour developper le prouveur. 
Nous avons utilisé l'outil Isabelle HOL pour developper la théorie. 

Le fichier tp89.thy contient le prouveur et le fichier executable permet
de voir l'execution.
